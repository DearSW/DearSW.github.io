基本命令集合

  • 让某个程序后台运行：使用nohup Command &命令，退出终端时，需要使用exit命令，来保证这个程序不被结束。
    • 如何结束这个程序？使用ps -ef | grep （程序相关的关键字）命令查看程序的pid，再通过kill 程序的pid命令来结束这个程序。
    • 值得一提的是，control + c这个操作是将程序暂时挂起，并没有将程序运行在后台！
